strava-lambda

Introduction

This is a project that tries to solve one of the big missing parts in the public Strava API: a webhook for updates of following athletes. I tried to solve this by using a lightweight Amazon Lambda function that polls the Strava API for new activities at set intervals and POSTs them to any API. The Lambda function and related S3 bucket can be deployed using Amazons Free Tier at no cost, but be aware that setting the updated interval in the Lambda function too high may result in additional charges (my advice: set a warning for when you are about to be charged).
